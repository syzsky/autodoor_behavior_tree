# 优化方案：随机范围功能设计

## 1 功能概述

### 1.1 需求背景

在自动化脚本执行过程中，固定的时间参数（如点击时长、延时时长、重复间隔等）容易被检测系统识别为机器行为。为了提高脚本的拟人化程度，需要为时间相关参数增加随机范围功能。

### 1.2 功能目标

为鼠标点击时长、按键时长、鼠标拖拽时长、节点的重复间隔等时间参数增加随机范围，每个节点的每个时长类参数和间隔类参数都可以单独设置随机范围，每次执行时根据基础值和随机范围选择一个随机值，使行为更加自然。

### 1.3 核心价值

| 价值 | 说明 |
|------|------|
| 拟人化 | 模拟人类操作的不规律性 |
| 反检测 | 避免固定时间模式被识别 |
| 灵活性 | 每个参数可单独设置随机范围 |
| 兼容性 | 不影响现有项目，向后兼容 |

---

## 2 参数梳理

### 2.1 设计原则

**每个节点的每个时长类参数和间隔类参数都可以单独设置随机范围**，互不影响。

### 2.2 适用参数列表

#### 2.2.1 动作节点参数

| 节点类型 | 参数名 | 参数说明 | 建议随机范围 |
|---------|--------|---------|-------------|
| KeyPressNode | duration | 按住时长(ms) | ±20ms |
| MouseClickNode | duration | 按住时长(ms) | ±20ms |
| MouseClickNode | click_interval | 点击间隔(ms) | ±30ms |
| MouseMoveNode | drag_duration | 拖拽时长(ms) | ±50ms |
| DelayNode | duration_ms | 延时时长(ms) | ±50ms |

#### 2.2.2 装饰器参数

| 参数名 | 适用节点类型 | 参数说明 | 建议随机范围 |
|--------|------------|---------|-------------|
| repeat_interval_ms | 所有动作节点 | 重复间隔(ms) | ±50ms |
| repeat_interval_ms | 所有复合节点 | 重复间隔(ms) | ±50ms |

#### 2.2.3 复合节点参数

| 节点类型 | 参数名 | 参数说明 | 建议随机范围 |
|---------|--------|---------|-------------|
| SequenceNode | child_interval | 子节点间隔(ms) | ±50ms |
| SelectorNode | child_interval | 子节点间隔(ms) | ±50ms |

### 2.3 参数分类

```
随机范围参数分类
├── 时长类参数
│   ├── duration (按住时长) - KeyPressNode, MouseClickNode
│   ├── duration_ms (延时时长) - DelayNode
│   └── drag_duration (拖拽时长) - MouseMoveNode
│
└── 间隔类参数
    ├── click_interval (点击间隔) - MouseClickNode
    ├── repeat_interval_ms (重复间隔) - 所有动作节点、复合节点
    └── child_interval (子节点间隔) - SequenceNode, SelectorNode
```

## 3 参数设计

### 3.1 随机范围参数格式

#### 3.1.1 命名规范

每个需要随机功能的参数，添加对应的 `{参数名}_random` 参数：

```python
{
    "duration": 100,           # 基础值
    "duration_random": 50      # 随机范围（±50ms）
}
```

#### 3.1.2 计算公式

```python
实际值 = random.randint(基础值 - 随机范围, 基础值 + 随机范围)
实际值 = max(0, 实际值)  # 确保不为负数
```

#### 3.1.3 示例

| 基础值 | 随机范围 | 最小值 | 最大值 | 可能的实际值 |
|--------|---------|--------|--------|-------------|
| 100 | 0 | 100 | 100 | 100 (固定) |
| 100 | 20 | 80 | 120 | 85, 103, 118, ... |
| 100 | 50 | 50 | 150 | 67, 124, 89, ... |
| 50 | 30 | 20 | 80 | 35, 71, 42, ... |

## 5 执行逻辑修改

### 5.1 随机值计算工具函数

```python
# bt_utils/helpers.py

import random
from typing import Union


def get_random_value(
    base_value: Union[int, float],
    random_range: Union[int, float] = 0,
    min_value: Union[int, float, None] = None
) -> Union[int, float]:
    """
    计算随机值
    
    Args:
        base_value: 基础值
        random_range: 随机范围（±值），默认为0（不随机）
        min_value: 最小值限制，默认为None（自动为0）
    
    Returns:
        随机后的值
    
    Examples:
        >>> get_random_value(100, 20)
        85  # 随机值在80-120之间
        
        >>> get_random_value(100, 0)
        100  # 无随机范围，返回基础值
        
        >>> get_random_value(50, 30, min_value=0)
        35  # 确保不小于0
    """
    if random_range <= 0:
        return base_value
    
    min_val = base_value - random_range
    max_val = base_value + random_range
    
    if min_value is not None:
        min_val = max(min_value, min_val)
    else:
        min_val = max(0, min_val)
    
    if isinstance(base_value, int):
        return random.randint(int(min_val), int(max_val))
    else:
        return random.uniform(min_val, max_val)


def get_random_duration(base_duration: int, random_range: int = 0) -> int:
    """
    计算随机时长（专用函数，确保时长不为负数）
    
    Args:
        base_duration: 基础时长(ms)
        random_range: 随机范围(±ms)
    
    Returns:
        随机后的时长(ms)
    """
    return get_random_value(base_duration, random_range, min_value=0)


def get_random_interval(base_interval: int, random_range: int = 0) -> int:
    """
    计算随机间隔（专用函数，确保间隔不为负数）
    
    Args:
        base_interval: 基础间隔(ms)
        random_range: 随机范围(±ms)
    
    Returns:
        随机后的间隔(ms)
    """
    return get_random_value(base_interval, random_range, min_value=0)
```

```python
from bt_utils.helpers import get_random_duration


class DelayNode(ActionNode):
    NODE_TYPE = "DelayNode"
    
    def __init__(self, node_id: str = None, config: NodeConfig = None):
        super().__init__(node_id, config)
        self.duration_ms = self.config.get_int("duration_ms", 1000)
        self.duration_ms_random = self.config.get_int("duration_ms_random", 0)
        self._start_time = None
        self._actual_duration = None
    
    def _execute_action(self, context) -> NodeStatus:
        import time
        
        if self._start_time is None:
            self._actual_duration = get_random_duration(self.duration_ms, self.duration_ms_random)
            self._start_time = time.time()
        
        elapsed = (time.time() - self._start_time) * 1000
        
        if elapsed < self._actual_duration:
            return NodeStatus.RUNNING
        
        return NodeStatus.SUCCESS
    
    def reset(self, reset_counters: bool = True) -> None:
        super().reset(reset_counters)
        self._start_time = None
        self._actual_duration = None
    
    def to_dict(self) -> Dict[str, Any]:
        data = super().to_dict()
        data["config"]["duration_ms"] = self.duration_ms
        data["config"]["duration_ms_random"] = self.duration_ms_random
        return data
```

```python
from bt_utils.helpers import get_random_interval


class ActionNode(Node):
    def _execute_with_decorators(self, context) -> NodeStatus:
        if self.config.repeat_count != 0:
            if self._repeat_count < abs(self.config.repeat_count) or self.config.repeat_count == -1:
                status = self._execute_action(context)
                
                if status == NodeStatus.SUCCESS:
                    self._repeat_count += 1
                    
                    if self.config.repeat_count == -1 or self._repeat_count < abs(self.config.repeat_count):
                        repeat_interval_random = self.config.get("repeat_interval_ms_random", 0)
                        actual_interval = get_random_interval(
                            self.config.repeat_interval_ms,
                            repeat_interval_random
                        )
                        time.sleep(actual_interval / 1000.0)
                        return NodeStatus.RUNNING
                
                return status
        
        return self._execute_action(context)
```

#### 5.4.1 修改 SequenceNode

```python
# bt_core/nodes.py

from bt_utils.helpers import get_random_interval


class SequenceNode(CompositeNode):
    def _tick_internal(self, context) -> NodeStatus:
        if not self.children:
            return NodeStatus.SUCCESS
        
        while self.current_index < len(self.children):
            child = self.children[self.current_index]
            
            if not child.config.enabled:
                self.current_index += 1
                continue
            
            status = child.tick(context)
            
            if status == NodeStatus.RUNNING:
                return NodeStatus.RUNNING
            
            if status == NodeStatus.FAILURE:
                self.current_index = 0
                return NodeStatus.FAILURE
            
            self.current_index += 1
            
            if self.child_interval > 0 and self.current_index < len(self.children):
                child_interval_random = self.config.get("child_interval_random", 0)
                actual_interval = get_random_interval(self.child_interval, child_interval_random)
                time.sleep(actual_interval / 1000.0)
        
        self.current_index = 0
        return NodeStatus.SUCCESS
```

```python
            return NodeStatus.FAILURE
        
        while self.current_index < len(self.children):
            child = self.children[self.current_index]
            
            if not child.config.enabled:
                self.current_index += 1
                continue
            
            status = child.tick(context)
            
            if status == NodeStatus.RUNNING:
                return NodeStatus.RUNNING
            
            if status == NodeStatus.SUCCESS:
                self.current_index = 0
                return NodeStatus.SUCCESS
            
            self.current_index += 1
            
            if self.child_interval > 0 and self.current_index < len(self.children):
                child_interval_random = self.config.get("child_interval_random", 0)
                actual_interval = get_random_interval(self.child_interval, child_interval_random)
                time.sleep(actual_interval / 1000.0)
        
        self.current_index = 0
        return NodeStatus.FAILURE
```

## 10 注意事项

1. **向后兼容**：确保旧版本项目能正常加载，缺失的随机参数自动设置为0
2. **性能影响**：随机值计算应尽量简单，避免影响执行效率
3. **可重复性**：如果需要可重复的执行结果，应提供禁用随机的选项
4. **日志记录**：建议在日志中记录实际使用的随机值，便于调试
5. **配置灵活性**：每个参数可单独设置随机范围，互不影响
6. **检测间隔不随机**：check_interval_ms 保持固定值，不添加随机功能

```python
    """使用随机种子的随机值计算"""
```

### 11.2 随机分布

当前使用均匀分布，未来可以支持其他分布：

```python
# bt_utils/helpers.py

import random
from enum import Enum


class RandomDistribution(Enum):
    UNIFORM = "uniform"      # 均匀分布
    NORMAL = "normal"        # 正态分布
    TRIANGULAR = "triangular"  # 三角分布


def get_random_value_with_distribution(
    base_value: int,
    random_range: int,
    distribution: RandomDistribution = RandomDistribution.UNIFORM
) -> int:
    """使用指定分布的随机值计算"""
    min_val = max(0, base_value - random_range)
    max_val = base_value + random_range
    
    if distribution == RandomDistribution.UNIFORM:
        return random.randint(min_val, max_val)
    elif distribution == RandomDistribution.NORMAL:
        value = random.gauss(base_value, random_range / 3)
        return int(max(min_val, min(max_val, value)))
    elif distribution == RandomDistribution.TRIANGULAR:
        value = random.triangular(min_val, max_val, base_value)
        return int(value)
```
