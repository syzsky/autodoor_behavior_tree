# 条件节点识别方向功能开发方案（精简版）

## 1 需求概述

### 1.1 功能需求

当前项目中 OCR 相关的条件节点（文字检测节点、数字比较节点），在识别到多个目标时，默认从左上往右下识别，返回识别到的第一个预期目标后返回坐标值。本需求旨在为这两个条件节点添加"识别方向"参数，允许用户指定识别的起始方向。

**涉及的节点**：
- **文字检测节点**（OCRConditionNode）
- **数字比较节点**（NumberConditionNode）

**支持的识别方向**：
- 左上（top-left）：默认行为，从左上角开始识别
- 右上（top-right）：从右上角开始识别
- 左下（bottom-left）：从左下角开始识别
- 右下（bottom-right）：从右下角开始识别

### 1.2 应用场景

| 场景 | 描述 | 推荐方向 |
|------|------|---------|
| 多个相同文字 | 屏幕上有多个相同的"登录"按钮，需要点击最右侧的 | 右上 或 右下 |
| 多个数字显示 | 屏幕上有多个倒计时数字，需要识别最下方的 | 左下 或 右下 |
| 列表界面 | 列表中有多个相同关键词，需要点击最上方的 | 左上 或 右上 |
| 动态界面 | 元素位置会变化，需要按特定方向优先识别 | 根据实际情况选择 |

## 2 现有架构分析

### 2.1 涉及的条件节点

| 节点类型 | 文件路径 | 识别引擎 | 当前行为 |
|---------|---------|---------|---------|
| OCRConditionNode | `bt_nodes/conditions/ocr.py` | OCRManager.recognize() | 按文本顺序遍历，返回第一个匹配 |
| NumberConditionNode | `bt_nodes/conditions/number.py` | OCRManager.recognize_number_with_position() | 返回第一个识别到的数字位置 |

### 2.2 当前识别逻辑详解

#### 2.2.1 OCRConditionNode

```python
# bt_utils/ocr_manager.py 第716-742行
for i, text in enumerate(result.txts):
    if not text:
        continue
    
    text_lower = text.lower()
    for keyword in keyword_list:
        keyword_idx = text_lower.find(keyword)
        
        if keyword_idx != -1:
            box = result.boxes[i]
            x, y = calculate_keyword_position(text, keyword, box)
            # 直接返回第一个匹配，无法控制方向
            return (True, (x, y), all_text)
```

**问题**：RapidOCR 返回的 `result.txts` 和 `result.boxes` 是按识别顺序排列的，通常是"从上到下、从左到右"的顺序。当屏幕上有多个相同关键词时，只能返回第一个匹配的位置。

#### 2.2.2 NumberConditionNode

```python
# bt_utils/ocr_manager.py 第842-865行
result = self._engine(img_array)
all_text = " ".join(result.txts)

# 只使用第一个文本框的位置
```

**问题**：只使用 `result.boxes[0]`，即第一个识别到的文本框位置。当区域内有多个数字时，无法选择特定位置的数字。

### 2.3 共同依赖

两个节点都依赖 `OCRManager` 的识别方法，核心修改点集中在此模块。

## 3 可行性评估

### 3.1 技术可行性

| 节点类型 | 可行性 | 实现难度 | 说明 |
|---------|--------|---------|------|
| OCRConditionNode | ✅ 可行 | ⭐⭐ 中等 | 需要收集所有匹配位置，按方向排序后返回 |
| NumberConditionNode | ✅ 可行 | ⭐⭐ 中等 | 需要收集所有数字位置，按方向排序后返回 |

### 3.2 实现难度分析

**优势**：
1. 两个节点共用同一个 OCR 引擎（OCRManager），修改点集中
2. RapidOCR 已返回所有识别结果（`result.boxes`），无需额外获取
3. 只需在现有逻辑基础上增加"收集 → 排序 → 返回"的流程

**挑战**：
1. 需要遍历所有识别结果，轻微增加处理时间
2. 需要处理坐标缩放和区域偏移

### 3.3 性能影响评估

| 操作 | 性能影响 | 说明 |
|------|---------|------|
| 收集所有匹配位置 | 轻微增加内存 | 通常匹配数量 < 10，影响可忽略 |
| 按方向排序 | O(n log n) | 匹配数量少，耗时 < 1ms |
| 缓存兼容 | 需要更新缓存键 | 将 `search_direction` 纳入缓存键 |

### 3.4 向后兼容性

**兼容策略**：默认值为"左上"，保持现有行为不变。

旧项目加载时：
- 未设置 `search_direction` → 使用默认值"左上"
- 行为与现有逻辑完全一致

```python
class SearchDirection:
    """识别方向常量"""
    TOP_LEFT = "top-left"      # 左上（默认）
    TOP_RIGHT = "top-right"    # 右上
    BOTTOM_LEFT = "bottom-left"  # 左下
    BOTTOM_RIGHT = "bottom-right"  # 右下
    
    # 中文显示映射
    DISPLAY_NAMES = {
        TOP_LEFT: "左上",
        TOP_RIGHT: "右上",
        BOTTOM_LEFT: "左下",
        BOTTOM_RIGHT: "右下",
    }
    
    # 反向映射（用于从配置读取）
    VALUE_MAP = {
        "左上": TOP_LEFT,
        "右上": TOP_RIGHT,
        "左下": BOTTOM_LEFT,
        "右下": BOTTOM_RIGHT,
    }
```

#### 4.1.2 位置排序函数

```python
# bt_utils/direction.py

from typing import List, Tuple

def sort_positions_by_direction(
    positions: List[Tuple[int, int]], 
    direction: str
) -> List[Tuple[int, int]]:
    """按识别方向排序位置列表
    
    Args:
        positions: 位置列表 [(x, y), ...]
        direction: 识别方向
        
    Returns:
        排序后的位置列表
    """
    if not positions:
        return positions
    
    if direction == SearchDirection.TOP_LEFT:
        # 左上：先按y升序（从上到下），再按x升序（从左到右）
        return sorted(positions, key=lambda p: (p[1], p[0]))
    
    elif direction == SearchDirection.TOP_RIGHT:
        # 右上：先按y升序（从上到下），再按x降序（从右到左）
        return sorted(positions, key=lambda p: (p[1], -p[0]))
    
    elif direction == SearchDirection.BOTTOM_LEFT:
        # 左下：先按y降序（从下到上），再按x升序（从左到右）
        return sorted(positions, key=lambda p: (-p[1], p[0]))
    
    elif direction == SearchDirection.BOTTOM_RIGHT:
        # 右下：先按y降序（从下到上），再按x降序（从右到左）
        return sorted(positions, key=lambda p: (-p[1], -p[0]))
    
    return positions
```

```python
def recognize(self, image: Image.Image, keywords: str = None,
              language: str = "eng",
              preprocess_mode: str = "normal",
              region: Tuple[int, int, int, int] = None,
              search_direction: str = "top-left",  # 新增参数
              use_cache: bool = True) -> Tuple[bool, Optional[Tuple[int, int]], str]:
    """执行OCR识别
    
    Args:
        image: PIL.Image 图像
        keywords: 关键词（逗号分隔）
        language: OCR语言
        preprocess_mode: 预处理模式
        region: 截图区域 (left, top, right, bottom)，用于坐标转换
        search_direction: 识别方向 (top-left/top-right/bottom-left/bottom-right)
        use_cache: 是否使用缓存
        
    Returns:
        (是否找到, 位置坐标, 所有识别文本) 元组
    """
    try:
        if not self._available:
            return False, None, ""
        
        if self._engine is None:
            return False, None, ""
        
        cache_key = None
        if use_cache:
            cache_key = self._compute_cache_key(
                image, keywords=keywords, language=language,
                preprocess_mode=preprocess_mode, region=region,
                search_direction=search_direction  # 新增：纳入缓存键
            )
            cached = self._get_cached_result(cache_key)
            if cached is not None:
                return cached
        
        processed = self._preprocess_image(image, language, preprocess_mode)
        
        img_array = np.array(processed)
        
        result = self._engine(img_array)
        
        if result is None:
            return False, None, ""
        
        if result.boxes is None or len(result.boxes) == 0:
            return False, None, ""
        
        all_text = " ".join(result.txts) if result.txts else ""
        
        if keywords:
            keyword_list = [k.strip().lower() for k in keywords.split(",")]
            
            # 【修改】收集所有匹配位置
            all_matches = []
            
            for i, text in enumerate(result.txts):
                if not text:
                    continue
                
                text_lower = text.lower()
                for keyword in keyword_list:
                    keyword_idx = text_lower.find(keyword)
                    
                    if keyword_idx != -1:
                        box = result.boxes[i]
                        
                        x, y = calculate_keyword_position(text, keyword, box)
                        
                        if image.size != processed.size:
                            scale_x = image.size[0] / processed.size[0]
                            scale_y = image.size[1] / processed.size[1]
                            x = int(x * scale_x)
                            y = int(y * scale_y)
                        
                        if region:
                            x += region[0]
                            y += region[1]
                        
                        all_matches.append((x, y))
            
            # 【新增】如果有匹配，按方向排序后返回第一个
            if all_matches:
                from bt_utils.direction import sort_positions_by_direction
                sorted_matches = sort_positions_by_direction(all_matches, search_direction)
                result_data = (True, sorted_matches[0], all_text)
                if cache_key:
                    self._set_cached_result(cache_key, result_data)
                return result_data
            
            result_data = (False, None, all_text)
            if cache_key:
                self._set_cached_result(cache_key, result_data)
            return result_data
        
        result_data = (True, None, all_text)
        if cache_key:
            self._set_cached_result(cache_key, result_data)
        return result_data
    
    except Exception as e:
        _logger.error(f"OCR识别异常: {e}", exc_info=True)
        return False, None, ""
```

```python
def recognize_number_with_position(self, image: Image.Image, language: str = "eng",
                                    preprocess_mode: str = "normal",
                                    extract_mode: str = "无规则",
                                    extract_pattern: str = "",
                                    min_confidence: float = 0.5,
                                    search_direction: str = "top-left",  # 新增参数
                                    use_cache: bool = True) -> Tuple[bool, Optional[float], str, Optional[Tuple[int, int]]]:
    """识别数字（带位置）
    
    Args:
        image: PIL.Image 图像
        language: OCR语言
        preprocess_mode: 预处理模式
        extract_mode: 提取模式 (无规则/x/y/自定义)
        extract_pattern: 自定义提取模式（使用*作为通配符）
        min_confidence: 最小置信度
        search_direction: 识别方向 (top-left/top-right/bottom-left/bottom-right)
        use_cache: 是否使用缓存
        
    Returns:
        (是否识别成功, 数字值, 所有识别文本, 位置坐标) 元组
    """
    try:
        if not self._available:
            return False, None, "", None
        
        if self._engine is None:
            return False, None, "", None
        
        cache_key = None
        if use_cache:
            cache_key = self._compute_cache_key(
                image, language=language, preprocess_mode=preprocess_mode,
                extract_mode=extract_mode, extract_pattern=extract_pattern,
                min_confidence=min_confidence, method="number",
                search_direction=search_direction  # 新增：纳入缓存键
            )
            cached = self._get_cached_result(cache_key)
            if cached is not None:
                return cached
        
        processed = self._preprocess_image(image, language, preprocess_mode)
        
        img_array = np.array(processed)
        
        result = self._engine(img_array)
        
        if result is None or result.txts is None or len(result.txts) == 0:
            return False, None, "", None
        
        all_text = " ".join(result.txts)
        
        # 【修改】收集所有包含数字的文本框位置
        all_number_data = []  # [(x, y, number_value), ...]
        
        if result.boxes is not None and len(result.boxes) > 0:
            for i, box in enumerate(result.boxes):
                # 计算文本框中心位置
                center_x = int((box[0][0] + box[2][0]) / 2)
                center_y = int((box[0][1] + box[2][1]) / 2)
                
                # 坐标缩放
                if image.size != processed.size:
                    scale_x = image.size[0] / processed.size[0]
                    scale_y = image.size[1] / processed.size[1]
                    center_x = int(center_x * scale_x)
                    center_y = int(center_y * scale_y)
                
                # 尝试从该文本框提取数字
                if result.txts and i < len(result.txts):
                    text = result.txts[i]
                    extracted = self._extract_number(text, extract_mode, extract_pattern)
                    if extracted is not None:
                        all_number_data.append((center_x, center_y, extracted))
        
        # 【新增】如果有数字，按方向排序后返回第一个
        if all_number_data:
            from bt_utils.direction import sort_positions_by_direction
            
            positions = [(d[0], d[1]) for d in all_number_data]
            sorted_positions = sort_positions_by_direction(positions, search_direction)
            
            # 找到排序后第一个位置对应的数字
            first_pos = sorted_positions[0]
            extracted_value = None
            for x, y, val in all_number_data:
                if x == first_pos[0] and y == first_pos[1]:
                    extracted_value = val
                    break
            
            result_data = (True, extracted_value, all_text, first_pos)
            if cache_key:
                self._set_cached_result(cache_key, result_data)
            return result_data
        
        result_data = (False, None, all_text, None)
        if cache_key:
            self._set_cached_result(cache_key, result_data)
        return result_data
    
    except Exception as e:
        _logger.error(f"OCR数字识别异常: {e}", exc_info=True)
        return False, None, "", None
```

**文件**：`bt_nodes/conditions/number.py`

```python
from bt_core.nodes import ConditionNode
from bt_core.config import NodeConfig
from typing import Dict, Any, Tuple, Optional
from bt_utils.ocr_manager import OCRManager


class NumberConditionNode(ConditionNode):
    NODE_TYPE = "NumberConditionNode"

    def __init__(self, node_id: str = None, config: NodeConfig = None):
        super().__init__(node_id, config)
        self.region: Optional[Tuple[int, int, int, int]] = self._parse_region(self.config.get("region", None))
        self.comparison = self.config.get("compare_mode", ">=")
        self.target_value = self.config.get_float("threshold", 0)
        self.extract_mode = self.config.get("extract_mode", "无规则")
        self.extract_pattern = self.config.get("extract_pattern", "")
        self.min_confidence = self.config.get_float("min_confidence", 50) / 100.0
        self.value_key = self.config.get("value_key", "last_number_value")
        
        # 新增：识别方向参数
        self.search_direction = self.config.get("search_direction", "左上")
        
        language_display = self.config.get("language", "简体中文")
        from bt_nodes.conditions.ocr import LANGUAGE_MAP
        self.language = LANGUAGE_MAP.get(language_display, "chi_sim")
        preprocess_display = self.config.get("preprocess_mode", "默认")
        self.preprocess_mode = "game" if preprocess_display == "复杂色彩" else "normal"

    def _check_condition(self, context) -> bool:
        try:
            if self.region is None:
                self._log_condition_result(False, "请先设置检测区域")
                return False

            screenshot = self._get_region_image(context)
            if screenshot is None:
                return False

            # 【修改】传入识别方向参数
            from bt_utils.direction import SearchDirection
            direction = SearchDirection.VALUE_MAP.get(self.search_direction, SearchDirection.TOP_LEFT)
            
            success, value, all_text, position = OCRManager().recognize_number_with_position(
                screenshot,
                language=self.language,
                preprocess_mode=self.preprocess_mode,
                extract_mode=self.extract_mode,
                extract_pattern=self.extract_pattern,
                min_confidence=self.min_confidence,
                search_direction=direction  # 新增
            )

            if not success or value is None:
                self._log_condition_result(False, f"无法识别数字 (文本: {all_text})")
                return False

            if position:
                actual_position = (position[0] + self.region[0], position[1] + self.region[1])
                self._save_position(context, actual_position)

            context.blackboard.set("last_number_value", value)
            if self.value_key and self.value_key != "last_number_value":
                context.blackboard.set(self.value_key, value)

            result = self._compare_value(value)

            if result:
                self._log_condition_result(True, extra_info=f"值: {value}")
                return True
            else:
                self._log_condition_result(False,
                    f"数值比较失败: {value} {self.comparison} {self.target_value}")
                return False
        except Exception as e:
            from bt_utils.exception_handler import log_exception
            log_exception(e, f"NumberConditionNode '{self.name}'")
            self._log_condition_result(False, "检测异常，详情见终端日志")
            return False

    def _compare_value(self, value: float) -> bool:
        """比较数值"""
        ops = {
            ">": lambda a, b: a > b,
            ">=": lambda a, b: a >= b,
            "<": lambda a, b: a < b,
            "<=": lambda a, b: a <= b,
            "==": lambda a, b: a == b,
            "!=": lambda a, b: a != b,
        }
        op = ops.get(self.comparison, lambda a, b: False)
        return op(value, self.target_value)

    def to_dict(self) -> Dict[str, Any]:
        data = super().to_dict()
        data["config"]["region"] = list(self.region) if self.region else None
        data["config"]["compare_mode"] = self.comparison
        data["config"]["threshold"] = self.target_value
        data["config"]["extract_mode"] = self.extract_mode
        data["config"]["extract_pattern"] = self.extract_pattern
        data["config"]["min_confidence"] = int(self.min_confidence * 100)
        data["config"]["value_key"] = self.value_key
        data["config"]["search_direction"] = self.search_direction  # 新增
        from bt_nodes.conditions.ocr import LANGUAGE_MAP
        reverse_language_map = {"eng": "English", "chi_sim": "简体中文", "chi_tra": "繁体中文"}
        data["config"]["language"] = reverse_language_map.get(self.language, self.language)
        data["config"]["preprocess_mode"] = "复杂色彩" if self.preprocess_mode == "game" else "默认"
        data["config"]["offset"] = list(self.offset)
        return data
```

```python
import pytest
from bt_utils.direction import sort_positions_by_direction, SearchDirection

class TestSortPositionsByDirection:
    
    def test_top_left(self):
        """测试左上方向：优先返回 y 最小、x 最小的位置"""
        positions = [(10, 10), (20, 5), (5, 20), (15, 15)]
        result = sort_positions_by_direction(positions, SearchDirection.TOP_LEFT)
        assert result[0] == (20, 5)   # y=5 最小
        assert result[1] == (10, 10)  # y=10 次小，x=10 < x=15
        assert result[-1] == (5, 20)  # y=20 最大
    
    def test_top_right(self):
        """测试右上方向：优先返回 y 最小、x 最大的位置"""
        positions = [(10, 10), (20, 5), (5, 20), (15, 15)]
        result = sort_positions_by_direction(positions, SearchDirection.TOP_RIGHT)
        assert result[0] == (20, 5)   # y=5 最小，x=20 最大
    
    def test_bottom_left(self):
        """测试左下方向：优先返回 y 最大、x 最小的位置"""
        positions = [(10, 10), (20, 5), (5, 20), (15, 15)]
        result = sort_positions_by_direction(positions, SearchDirection.BOTTOM_LEFT)
        assert result[0] == (5, 20)   # y=20 最大，x=5 最小
    
    def test_bottom_right(self):
        """测试右下方向：优先返回 y 最大、x 最大的位置"""
        positions = [(10, 10), (20, 5), (5, 20), (15, 15)]
        result = sort_positions_by_direction(positions, SearchDirection.BOTTOM_RIGHT)
        assert result[0] == (5, 20)   # y=20 最大
    
    def test_empty_positions(self):
        """测试空列表"""
        result = sort_positions_by_direction([], SearchDirection.TOP_LEFT)
        assert result == []
```

## 9 总结

本开发方案针对 OCR 相关条件节点（文字检测、数字比较）设计了识别方向功能，主要结论如下：

### 9.1 方案优势

1. **修改范围小**：只涉及 2 个节点，修改集中在 `OCRManager` 模块
2. **实现难度低**：无需修改底层识别引擎，只需增加排序逻辑
3. **性能影响小**：匹配数量通常较少，排序开销可忽略
4. **向后兼容**：默认值为"左上"，保持现有行为不变

### 9.2 开发工时

预估总工时为 **6 小时**，可在 **1 个工作日**内完成。

### 9.3 实施建议

建议按照本方案的实施计划逐步推进：
1. 先完成基础设施（direction.py）
2. 再修改核心组件（OCRManager）
3. 最后修改节点层和 GUI 层

功能稳定后，可考虑后续扩展到图像匹配节点和颜色检测节点。
