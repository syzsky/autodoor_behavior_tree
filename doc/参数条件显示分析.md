# GUI参数条件显示分析文档

本文档记录了所有节点中存在参数依赖关系的情况，即当某个参数选择特定值时，其他参数无需填写或自动隐藏。

---

## 1. 按键节点 (KeyPressNode) - 已实现

### 参数依赖关系

| 条件参数 | 条件值 | 隐藏/无效参数 | 原因 |
|---------|--------|--------------|------|
| action | down | duration, duration_random | down动作只按下按键，不涉及时长 |
| action | up | duration, duration_random | up动作只释放按键，不涉及时长 |

### 实现代码

```python
"KeyPressNode": [
    {"key": "key", "label": "按键", "type": "key"},
    {"key": "action", "label": "动作", "type": "select", "options": ["press", "down", "up"], "default": "press"},
    {"key": "duration", "label": "按住时长(ms)", "type": "number", "default": 0, "hide_if": {"field": "action", "value": ["down", "up"]}},
    {"key": "duration_random", "label": "时长随机范围(±ms)", "type": "number", "min": 0, "default": 0, "hide_if": {"field": "action", "value": ["down", "up"]}},
],
```

## 技术实现

### hide_if 多值支持

在 `property.py` 的 `_update_single_field_visibility` 方法中扩展了 `hide_if` 支持列表值：

```python
def _update_single_field_visibility(self, key: str, field: Dict[str, Any]):
    hide_if = field.get("hide_if")
    if not hide_if:
        return
    
    depend_field = hide_if.get("field")
    hide_value = hide_if.get("value")
    
    # ...
    
    current_value = depend_widget.get_value()
    
    # 支持多值判断
    if isinstance(hide_value, list):
        should_hide = current_value in hide_value
    else:
        should_hide = (current_value == hide_value)
    
    # ...
```
