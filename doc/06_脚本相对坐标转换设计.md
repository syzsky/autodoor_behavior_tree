# 脚本相对坐标转换功能设计

## 1. 功能概述

为脚本节点（ScriptNode）提供坐标转换功能，包括：
1. **转换按钮**：将录制时保存的屏幕绝对坐标转换为窗口相对坐标
2. **执行时转换**：脚本执行时将窗口相对坐标转换回屏幕绝对坐标

## 3. 脚本文件格式扩展

### 3.1 原格式

```
Delay 100
MoveTo 100, 200
LeftDown 1
LeftUp 1
```

### 3.2 转换后格式

```
# Window: 记事本 - 无标题
# WindowRect: 100, 200, 800, 600
Delay 100
MoveTo 50, 100
LeftDown 1
LeftUp 1
```

### 3.3 窗口标记说明

| 标记 | 说明 |
|------|------|
| `# Window: <窗口标题>` | 记录转换时的窗口标题关键字 |
| `# WindowRect: <left, top, right, bottom>` | 记录转换时的窗口矩形区域（仅供参考） |

### 3.4 脚本运行兼容性

`# Window:` 和 `# WindowRect:` 开头的行会被 `ScriptExecutor` 忽略（因为不是有效命令），不影响脚本运行。

## 5. 坐标转换逻辑

### 5.1 屏幕绝对坐标 → 窗口相对坐标（转换按钮）

**使用方法**：`CoordinateConverter.absolute_to_window(x, y, hwnd)`

```python
from bt_utils.coordinate import CoordinateConverter

# 转换单个坐标
window_x, window_y = CoordinateConverter.absolute_to_window(
    screen_x, screen_y, hwnd
)

# 转换公式
window_x = screen_x - window_rect[0]  # 减去窗口左边界
window_y = screen_y - window_rect[1]  # 减去窗口上边界
```

### 5.2 窗口相对坐标 → 屏幕绝对坐标（脚本执行）

**使用方法**：`context.convert_to_screen_coords((x, y))` 或 `CoordinateConverter.window_to_absolute(x, y, hwnd)`

```python
# 方法一：通过 ExecutionContext
screen_x, screen_y = context.convert_to_screen_coords((window_x, window_y))

# 方法二：直接使用 CoordinateConverter
from bt_utils.coordinate import CoordinateConverter
screen_x, screen_y = CoordinateConverter.window_to_absolute(
    window_x, window_y, hwnd
)

# 转换公式
screen_x = window_x + window_rect[0]  # 加上窗口左边界
screen_y = window_y + window_rect[1]  # 加上窗口上边界
```

### 10.2 修改 _start_script 方法

```python
def _start_script(self, absolute_script_path, script_path, context):
    # 读取脚本内容
    with open(absolute_script_path, 'r', encoding='utf-8') as f:
        content = f.read()
    
    # 检查窗口标记
    marker = self._parse_window_marker(content)
    
    if marker["has_marker"]:
        # 有窗口标记，需要转换坐标
        bound_window = context.get_bound_window()
        if bound_window:
            content = self._convert_to_absolute_coords(content, context)
            print(f"[DEBUG] ScriptNode: 已将窗口相对坐标转换为屏幕绝对坐标")
    
    # 执行脚本
    self._executor.run_script(content, loop=use_loop)
```

## 11. 注意事项

### 11.1 窗口位置变化

如果窗口移动了，之前转换的相对坐标仍然有效，因为：
- 转换按钮：将屏幕绝对坐标转换为窗口相对坐标（保存到文件）
- 脚本执行：将窗口相对坐标转换为当前窗口位置的屏幕绝对坐标

### 11.2 多次转换保护

通过窗口标记防止重复转换，避免坐标被多次减去窗口偏移。

### 11.3 脚本运行兼容性

注释行（以 `#` 开头）会被 `ScriptExecutor` 自动忽略，不影响脚本运行。

### 11.4 未绑定窗口时的行为

如果脚本有窗口标记但行为树未绑定窗口：
- 脚本会直接使用窗口相对坐标作为屏幕绝对坐标执行
- 这可能导致鼠标移动到错误的位置
- 建议用户确保行为树已绑定窗口
