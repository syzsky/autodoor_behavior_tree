**v1.6.0 新增内容**

- **游戏自动化挂机模块**：完整 AFK 自动化流程，支持窗口绑定→检测起点→寻找NPC→传送→校验地图→自动挂机→背包回收→药品购买→A*卡怪寻路
- **YOLO 实时检测覆盖**：在实时预览画面叠加 YOLO 检测框，支持模型加载开关
- **运行时依赖自动下载**：exe 启动时自动检查并下载缺失的 Python 包和模型文件

**优化内容**

- **优化默认输入文本的输入间隔为0**：输入间隔为0可以直接将整段文本粘贴到输入框，从而无需等待输入，如果需要模拟真实输入，可以再修改输入间隔
- **exe 体积优化**：将 PyTorch/torchvision/ultralytics/scipy 从 exe 中排除，改为运行时自动安装，exe 从 313MB 降至 ~110MB

**修复内容**

- **尝试修复绑定窗口时，检测区域略微偏移的可能问题**
- **修复了图像识别置信度没有正确转换成百分数进行对比的问题**
- **修复组合节点失败重试时的首个节点状态会异常直接成功的问题**
